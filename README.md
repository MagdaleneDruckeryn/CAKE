CAE模型输入参数量化标准说明书
1. 基础物理控制参数
音高 (Pitch): 提供满足MIDI标准的整数值，范围为0到127。
力度 (Velocity): 提供归一化浮点数，范围为0.0到1.0。
时值 (Duration): 提供以秒为单位的精确时长数值。

2. 时域包络特征 (Time-Domain Features)
起音时间 (Attack Time): 提供以毫秒为单位的数值。
衰减速率 (Decay Rate): 提供以分贝每秒为单位的下降斜率。
能量变化曲线: 提供归一化后的幅值包络采样点序列。
RMS能量: 提供音频段的均方根能量平均值。

3. 频域与音色特征 (Spectral Features)
基频 (F0): 提供精确至 $0.1Hz$ 的基频频率数值。
谐波分布向量 (Harmonic Profile): 提供各次谐波相对于总能量的比例系数向量。
明亮度 (Brightness): 提供频谱质心的中心频率数值。
高频衰减特性: 提供频谱滚降点频率数值。
非谐波成分比例: 提供0.0到1.0之间的噪声比率数值。

4. 训练真值与标准化数据 (Ground Truth)
音频规格: 提供采样率固定为44.1kHz或48kHz的WAV文件。
声纹图特征: 提供短时傅里叶变换处理后的2D频谱矩阵。
尺寸标准化: 频谱矩阵需统一调整为 $256 \times 256$ 像素尺寸。
打击分量: 利用HPSS算法提取的用于刻画起音瞬间的物理分量。
时间对齐标记: 提供每个音频样本起音点的精确采样偏移量。

5. 教学对比量化指标
参数欧氏距离: 提供学生特征向量与标准特征向量的多维距离计算结果。
包络拟合误差: 量化学生演奏曲线与标准ADSR曲线的偏离程度。



CAE模型期望输出量化标准说明书


### 1. 最终音频输出 (Time-Domain Waveform)
该输出是系统最直观的成果，用于提供标准音色示范。

文件格式: 单声道 WAV 格式。
采样率: 固定为 44.1kHz 或 48kHz。
位深度: 16-bit 或 24-bit PCM 编码。
时值长度: 与输入参数 $duration$ 严格对齐。
物理特性: 波形需完整包含起音、衰减、持音与释音（ADSR）四个阶段。


### 2. 中间频谱输出 (Reconstructed Spectrogram)
该输出用于验证模型在频域上的重建精度，是生成音频的前置步骤。

数据结构: 2D 浮点数矩阵（Tensor）。
分辨率: 统一为 256 像素乘以 256 像素。
数值含义: 矩阵各点代表对应频率与时间点上的分贝强度（dB）或幅值。
频率分布: 必须清晰呈现基频（F0）及其对应的各级谐波结构。


### 3. 潜在空间参数映射 (Latent Space Mapping)
用于教学辅助中的偏差量化分析。

输出内容: 学生音频经过编码器后得到的参数向量 $\theta$。
量化维度: 包含音高、力度、明亮度等维度的标准化数值。
对比指标: 计算生成的标准 $\theta$ 与学生 $\theta$ 之间的多维欧氏距离。
差异反馈: 输出具体的数值差值，例如：力度偏差值或起音时间延迟毫秒数。



### 4. 模型性能评估指标 (Evaluation Metrics)
用于衡量 CAE 模型的训练质量与参数可控性。

重建损失 (Reconstruction Loss): 计算输出频谱与原始真值频谱之间的均方误差（MSE）。
参数一致性评分: 改变输入参数（如 $brightness$）时，输出频谱对应特征变化的线性相关度。
音频稳定性评分: 检测生成波形是否含有神经网络常见的电音伪影或不自然的相位噪声。
